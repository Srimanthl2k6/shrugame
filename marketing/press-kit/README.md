# Shrugame Press Kit

## One-Line Description

A weird Pacific Northwest pixel RPG where Shrububu searches five cursed districts for KFC and grows powerful enough to challenge demon king SRMT.

## Facts

- Genre: narrative pixel RPG / turn-based bullet-dodging encounters
- Engine: Godot 4.7 inside Electron
- Platforms: Windows x64; macOS Intel and Apple silicon
- Modes: Shrububu (extremely easy), SRMT (extremely hard)
- Players: one
- Save slots: one
- Version: 1.0.0

## Approved Media

Use the release screenshots and key art in `site/public/media/`. Do not use anything from private reference folders or `source_art/`.

## Short Description

Shrububu enters Ishiville looking for KFC. A disappointed door slam reveals impossible strength, and five strange districts recruit her to uncover clues, collect weapons, defeat ten bosses, free former queen IshiYoga, and confront SRMT before he finishes the bucket.
