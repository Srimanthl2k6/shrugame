# System Requirements

## Windows

- Windows 10 or 11, 64-bit
- Dual-core x64 processor
- 4 GB RAM
- WebGL 2-capable integrated or dedicated graphics
- 700 MB available storage
- Keyboard or compatible controller

## macOS

- macOS 12 or newer
- Intel x64 or Apple silicon
- 4 GB RAM
- 700 MB available storage
- Keyboard or compatible controller

## Display

- Minimum window: 960x540
- Default window: 1280x720
- Tested targets: 960x540, 1280x720, and 1920x1080
- The game renders internally at 640x360 with nearest-neighbor integer scaling.

Version 1.0.5 packages are unsigned. Windows SmartScreen or macOS Gatekeeper may require an explicit first-launch confirmation.
