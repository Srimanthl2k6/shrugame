# Shrugame 1.0.1

Version 1.0.1 is a focused usability and progression update.

## Fixed

- Replaced tiny enlarged pixel body text with readable Atkinson Hyperlegible text rendered at the native 640x360 UI resolution.
- Added first-play controls, movement, interaction, battle-command, and dodge tutorials. Tutorials follow remapped keyboard/controller bindings and can be replayed from the pause menu.
- Fixed the route after defeating Satyaki. Walking east now automatically enters Banana-burbs, updates progression, and atomically saves Level 2.
- Applied the same validated transition path to every district boundary through Area 111.
- Removed all constant music and ambience. Only non-looping interface, battle, weapon, cutscene, save, and transition SFX remain.

## Compatibility

- Existing 1.0.0 saves remain compatible.
- Existing saves made after defeating Satyaki immediately receive the working Level 2 exit.
- Difficulty, story flags, inventory, clues, gear, and boss progress are unchanged.

## Downloads

- Windows x64 installer and portable executable.
- Unsigned macOS Intel and Apple-silicon DMG/ZIP packages.
- SHA-256 checksum and artifact manifests.

These builds are unsigned. Windows SmartScreen or macOS Gatekeeper may display a first-launch warning.
