# Shrugame 1.0.0

Shrububu arrives in Ishiville looking for KFC and finds five cursed districts, ten fights, an imprisoned queen, and SRMT eating the evidence.

## Included

- Five connected-room districts and a complete ending.
- Shrububu and SRMT difficulty modes.
- Keyboard/controller play with remapping.
- Flash reduction, shake control, high-contrast bullets, text speed, objectives, and independent volume controls.
- One save file with migration, atomic writes, and corruption fallback.
- Windows x64 installer and portable app.
- macOS Intel and Apple-silicon DMG/ZIP packages.

## Unsigned Build Notice

This first release is unsigned. Windows SmartScreen and macOS Gatekeeper may display a warning. SHA-256 checksums are attached to the release so downloaded files can be verified.

Shrugame is an independent parody project and is not affiliated with KFC or any other referenced brand.
